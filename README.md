# password-generator
A basic password generator from Python that generates a strong password in the terminal using ascii leters, digits, and punctuation markers.

Some planned features are to be added in the future, such as the option for passwords to be saved as a text file, and for that to be encoded in hex, base64, XOR, binary, etc.

To run the code, download the file, locate it in your computer, then run it in your command line terminal.

For example, when using the linux terminal, run the command "python3 password-generator.py" and the code will run.
Use the program as you please, feel free to modify and change it as you wish.
